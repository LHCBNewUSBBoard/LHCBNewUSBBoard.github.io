1. Introduction

This article contains all the informations about the New USBBoard for SiPM detectors readout, including the basic introductions about the SiPM detectors, the development process, the operation manual and related bug issues. This article is also the summary of my three months work in EPFL.

If you have any problem, please contact hongming.

  • System Structure - Basic informations about the SiPM detectors & its readout.
  • Development Process - The introduction of the readout board, includes hardware, firmware and software.
  • Operation Manual - The operations needed to set up an experiment and the usage of all tools.
  • Bug Issues & Freq Asks- Record all the bugs happened in the experiments and the methods of solving them.
